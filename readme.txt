Sample NodeJs based Extjs grid app using Mongoose and MongoDB.  Grid exmaple includes CRUD, paging and sorting.

Tested with:
 * NodeJS v0.2.0+
 * Conect v0.2.4+
 * Mongoose  v0.0.2+
 * MongoDB v1.6.0

Deployment folder structure:

+ grid-test
  + lib
    + symlink to connect
  + public
    + ext-3.2.1 (download here)
    + shared
      + icons
        + fam (download famfamfam silk icons here)
      + js
        + ext-ux
